A premium, responsive Spotify Web Player front-end clone built using semantic HTML5 and vanilla CSS. This project replicates the user interface of Spotify's web browser player, utilizing curated components, Google Fonts, and modern design principles.

---

## 🎨 Design & Aesthetics

This clone prioritizes visual excellence and a premium experience:
- **Responsive Layout**: Adapts smoothly to various screen sizes.
- **Glassmorphism & Gradients**: Premium gradient notification bar mimicking Spotify's premium prompts.
- **Micro-interactions**: Hover transitions on cards, buttons, and navigation links.
- **Typography**: Uses the Baloo Bhai 2 and Material Symbols from Google Fonts for a clean, modern aesthetic.

---

## 🚀 Features

- **Sidebar Navigation**: Quick access to Home, Search, and Library.
- **Custom Playlists & Podcasts Sections**: Interactive placeholder cards for creating playlists and browsing podcasts.
- **Categorized Music Sections**: Organized grids for:
  - Popular Artists (Pritam, Arijit Singh, A.R. Rahman, etc.)
  - Popular Albums
  - Featured Charts
  - Trending Episodes
- **Persistent Player Preview**: Bottom bar with promotional sign-up text and CTA button.
